ETIQUETAS BASICAS DE HTML

<html>...</html>

La etiqueta <html>delimita y engloba toda la página web, que consta de cabecera y cuerpo.

<h1>...</h1>

<h1>Delimita y engloba la cabecera de una página, que contiene un conjunto de informaciones que no se muestran en la ventana,
entre ellas el título de la página, pero que pueden ayudar a los navegadores y a los buscadores para interpretar o a encontrar correctamente la página

<p>...</p>

Esta etiqueta <p> sirve para empezar y terminar un parrafo, se puede usar multiples veces

<span>...</span>

la etiqueta <span> se usa para resaltar alguna parte de un texto, con un color

<div>...</div>

Esta etiqueta <div> nos permite cambiar el color de fondo un una determinada linea de texto.

<ul>...<ul>

con esta etiqueta <ul> podemos organizar texto en forma de lista, debemos usar la etiqueta <li> para cada elemento de la lista.

Ejemplo:


<!DOCTYPE html>
<html>
<body>
<title>Titulo de Pagina</title>
<h1>cabecera</h1>
<p> <div>Este es un parrafo. </div></p>
<p> Este es otro parrafo.</p>
<ul>
<li>articulo 1</li>
<li>articulo 2</li>
<li>articulo 3</li>
</ul>
</body>
</html>
